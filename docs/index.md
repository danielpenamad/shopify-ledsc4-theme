# Documentación LedsC4 B2B

!!! warning "Documentación en construcción"
    **v0.1** publicada el 12-may-2026. Esta documentación está en evolución activa; varios documentos están todavía como esqueleto o parcial. Consulta la [tabla de estado completa](https://github.com/danielpenamad/shopify-ledsc4-theme/blob/main/docs/README.md) en el repositorio para ver el detalle.

Este sitio reúne la documentación técnica, de administración y de operación del proyecto **LedsC4 B2B**, una tienda Shopify Plus con flujos B2B (alta de profesionales, whitelist, multi-idioma, multi-divisa, integraciones con Supabase y pipeline de datos propio).

Está pensada para tres audiencias bien diferenciadas. Empieza por la sección que corresponda a tu rol.

## Secciones

| Sección | Para quién | Descripción |
| --- | --- | --- |
| [Desarrollo](desarrollo/index.md) | Equipo técnico | Arquitectura, pipeline de datos, theme Shopify, integraciones (Supabase, GitHub Actions), i18n y currency. |
| [Administración](administracion/index.md) | Administrador del negocio | Gestión diaria del catálogo, categorías, emails y traducciones desde el admin de Shopify. |
| [Operador](operador/index.md) | Operador de back-office | Flujo del día a día: aprobar altas de profesionales, gestionar whitelist y resolver incidencias. |

## Cómo navegar

- Usa la barra superior para cambiar de sección.
- El buscador (arriba a la derecha) cubre todo el sitio.
- Cada documento incluye una cabecera con su **estado** (✅ completo / ⚠️ parcial / 🚧 esqueleto) y su **audiencia**.
- Si un documento está bloqueado por trabajo pendiente, aparece indicado en la cabecera.
